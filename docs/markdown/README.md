# 常用写法

## 一：代码块高亮

在 .md 文件中书写代码时，可在 ``` 后增加 js、html、json 等格式类型，代码块即可按照指定类型高亮

```js
export default {
  data() {},
};
```

## 二：自定义容器

::: tip 提示
this is a tip
:::

::: warning 注意
this is a tip
:::

::: danger 警告
this is a tip
:::
