# typescript

```js
npm install -g typescript   //全局安装

npm install -D typescript   //当前项目安装

tsc init   //tsconfig.json

tsc a.ts   //编译  会生成一个同名的js文件

tsc -w     //每当文件保存就会自动编译
```
