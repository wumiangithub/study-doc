## 面向对象

面向对象由三部分组成：(封装、继承、多态) 有人说四部分：还有一个就是抽象  
封装：类是一种封装，方法是一种封装  
继承：子类继承父类  
多态：父类定义一个方法，没有具体实现。需要子类重写父类中的方法，这样不同的子类实例调用同样的方法会有不同的效果
